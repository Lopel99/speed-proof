Introduction

SpeedProof is a comprehensive GPS-based tracking application designed to monitor speed, location, and environmental conditions during various movement activities. The app serves two primary modes: Driving Mode for vehicle speed tracking and Hiking Mode for outdoor activities and trail documentation.

Key Features

  • Real-time GPS speed and location tracking
  • Weather condition monitoring
  • Waypoint marking and trail documentation
  • Emergency GPS coordinate sharing
  • Comprehensive activity reports
  • Safety controls and data export

Important Note

SpeedProof is designed for personal awareness and activity documentation. GPS accuracy can vary based on environmental conditions and device capabilities. The app is intended for informational purposes and personal use.

Getting Started

System Requirements

  • Android device with GPS capability
  • Location permissions enabled
  • Internet connection for weather data (optional for basic tracking)

First Launch

  1. Grant location permissions when prompted
  2. Allow the app to access your current location
  3. Weather data will automatically load for your area
  4. Choose between Driving Mode or Hiking Mode
  5. Press "Start Tracking" to begin monitoring

Button Functions Guide

Primary Controls

Start Tracking / Stop Tracking

  • Function: Initiates or terminates GPS monitoring
  • Behavior: Records location, speed, and timestamp data every 1-2 seconds
  • Status: Green when stopped, Red when active
  • Lock Protection: Disabled when controls are locked to prevent accidental stops

LOCK / UNLK

  • Function: Safety mechanism to prevent accidental button presses
  • Purpose: Particularly useful during driving to avoid ending tracking sessions
  • Visual: Orange when locked, Gray when unlocked
  • Toast Notification: Confirms lock status changes

ON / OFF (Display Control)

  • Function: Controls screen timeout behavior
  • ON: Keeps screen active during tracking (useful while driving)
  • OFF: Allows normal screen timeout to conserve battery
  • Visual: Green when ON, Gray when OFF

Mode Selection

Driving Mode

  • Purpose: Optimized for vehicle speed tracking
  • Display: Shows current speed in MPH with maximum speed tracking
  • Features: Speed records, trip documentation, backtrack reports
  • Use Cases: Personal driving awareness, trip documentation

Hiking Mode

  • Purpose: Designed for walking, hiking, and outdoor activities
  • Display: Shows GPS points collected and distance traveled
  • Features: Waypoint marking, emergency GPS, trail reports
  • Use Cases: Hiking, walking, outdoor recreation, emergency preparedness

Weather Functions

Refresh (Weather)

  • Function: Updates current weather data for your location
  • Data Source: OpenWeatherMap API
  • Information: Temperature, conditions, humidity, wind, visibility

Test Weather API

  • Function: Diagnostic tool to verify weather service connectivity
  • Purpose: Troubleshooting weather data issues
  • Fallback: Uses test coordinates if location unavailable

Hiking Mode Specific

Mark Waypoint

  • Function: Saves current GPS location as a named reference point
  • Information Captured: Coordinates, elevation, timestamp, automatic naming
  • Requirements: Must be actively tracking to mark waypoints
  • Use Cases: Points of interest, trail markers, emergency references

Emergency GPS

  • Function: Instantly generates shareable coordinate information
  • Output: Exact coordinates, elevation, weather conditions, emergency contact format
  • Purpose: Search and rescue, emergency situations, location sharing
  • Format: Compatible with professional GPS equipment and rescue services

Troubleshooting

Common Issues and Solutions

GPS Not Working

  • Ensure location permissions are granted
  • Move to area with clear sky view
  • Restart app and allow time for GPS lock
  • Check device GPS settings

Weather Data Not Loading

  • Verify internet connection
  • Use "Test Weather API" button for diagnostics
  • Weather functionality is optional - tracking works without it
  • Try refreshing weather data manually

Inaccurate Speed Readings

  • GPS accuracy varies by environment and device
  • Allow time for GPS to stabilize after app start
  • Avoid use in areas with poor GPS reception
  • Remember SpeedProof is for personal awareness, not precision measurement

App Crashes or Freezes

  • Restart the application
  • Ensure sufficient device storage space
  • Close other resource-intensive apps
  • Update to latest version if available

Battery Drain

  • Adjust screen timeout settings using ON/OFF button
  • Close app when not actively tracking
  • Consider device power saving modes for extended use

Performance Optimization

For Best GPS Performance

  • Use device with clear view of sky
  • Allow app to run in foreground during tracking
  • Avoid rapid movement between indoor/outdoor environments
  • Ensure device has adequate battery charge

For Extended Battery Life

  • Use OFF setting for display timeout
  • Track only when needed
  • Close app completely when finished
  • Consider airplane mode with GPS enabled in remote areas

Privacy & Data

Data Collection and Usage

SpeedProof collects only the minimum data necessary for functionality:

  • Location Data: GPS coordinates for speed and position calculation
  • Weather Data: Environmental conditions from third-party services
  • Usage Data: Anonymous app performance information

Data Storage and Control

  • Local Storage: All personal data remains on your device
  • No Cloud Storage: Location data is never transmitted to external servers
  • User Control: Complete ability to delete data at any time
  • No Account Required: No personal information collection

Accuracy Note

  • GPS signals affected by environmental factors
  • Weather data provided by third-party services
  • All data should be considered informational only
  • No warranty of accuracy or reliability provided

User Responsibilities

  1. Follow all applicable laws while using SpeedProof
  2. Do not use while driving unless safely mounted and configured
  3. Respect privacy of others when sharing location data
  4. Use emergency features responsibly and only when appropriate
  5. Understand limitations of GPS and weather data accuracy

SpeedProof: Movement & Activity Tracker provides comprehensive GPS monitoring for both vehicle and outdoor activities. By understanding the features, limitations, and proper usage guidelines outlined in this manual, users can maximize the app's value while maintaining safety and privacy.
