?? Understanding Compass Mode

?? Important: Your phone has a built-in digital compass - no physical compass required!
1 Enable Compass Mode

In Hiking Mode, tap "Compass Mode" to activate the digital compass display.

2 Hold Phone Correctly

Keep your phone flat and level for accurate compass readings. The app shows real-time bearing in degrees and cardinal directions.

3 How It Works

The compass uses your phone's sensors to show which direction you're facing. This helps with navigation and orientation.

?? Return to Start Feature

1 Keep Tracking Active

You do NOT need to stop tracking to use Return to Start. Keep tracking running for best results.

2 Press Return to Start

Tap the "Return to Start" button anytime during your hike to get navigation back to your starting point.

3 Follow Directions

The app provides distance, compass bearing, and estimated walking time back to your start location.

?? Pro Tip: Compass Mode and Return to Start work independently. You can use Return to Start with or without Compass Mode enabled.

? Frequently Asked Questions

Do I need a physical compass to use Compass Mode?

No! Your phone has a built-in digital compass using magnetometer sensors. No physical compass is required.

Do I need to stop tracking to use Return to Start?

No! Keep tracking active. Return to Start works while tracking is running and provides live navigation.

Does Compass Mode need to be on for Return to Start?

No! Return to Start works independently. Compass Mode is optional but can help you follow the bearing.

Why is my compass sometimes inaccurate?

Magnetic interference from electronics or metal objects can affect accuracy. Calibrate by moving phone in figure-8 pattern.

Does compass work without cell service?

Yes! The compass uses phone sensors, not cellular data. GPS may be slower without data but still works.
