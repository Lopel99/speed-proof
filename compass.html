? Frequently Asked Questions

Do I need a physical compass to use the navigation features?

No! The hybrid system provides continuous navigation on your phone screen. The physical compass option is available for battery conservation or as a backup.

How often does the navigation update?

The hybrid navigation updates every 2 seconds, showing real-time bearing, direction, and distance to your start point.

Do I need to stop tracking to use Return to Start?

No! Keep tracking active. The hybrid navigation works while tracking is running and provides live navigation updates.

Does the navigation work without cellular signal?

Yes! Once started, the hybrid navigation works completely offline using GPS and phone sensors only.

Why would I use a physical compass if phone navigation works?

Physical compass navigation conserves battery, works in extreme conditions, and provides reliable backup if your phone fails.

Can I use both navigation methods at the same time?

Yes! The hybrid system is designed to let you switch between phone display and physical compass as needed.
