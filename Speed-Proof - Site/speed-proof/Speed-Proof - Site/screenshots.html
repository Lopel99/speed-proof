Explore the SpeedProof Interface

Get a detailed look at SpeedProof's intuitive design and powerful features. Our dual-mode interface provides everything you need for accurate GPS tracking, whether you're driving or hiking.

Dashboard View

The main dashboard displays your current speed in real-time with clear, easy-to-read metrics.

  • Current Speed Display: Large, clear speed readout in MPH
  • Max Speed Tracking: Records your maximum speed during the session
  • Record Counter: Shows number of GPS data points collected
  • Status Indicator: Visual confirmation of tracking state (STOPPED/TRACKING)
  • Weather Integration: Real-time weather data loading for environmental context
  • Quick Actions: Start Tracking and Test Weather buttons for instant control

Control Panel

Access advanced features, report generation, and app settings through the comprehensive control panel.

  • Report Generation: Create legal documentation with weather data
  • 5-Mile Backtrack: Generate detailed route analysis reports
  • Share Reports: Easily share activity summaries via any app
  • Debug Information: Access technical diagnostics when needed
  • Mode Switching: Toggle between Driving and Hiking modes
  • Weather Controls: Manual weather refresh and API testing
